Getting Started
🔗

Channel Linking

Link channels between Discord and Fluxer to create seamless cross-platform conversations.

💬

Message Synchronization

Messages sent in linked channels are automatically mirrored between Discord and Fluxer in real time.

✏️

Edit & Delete Sync

Message edits and deletions propagate across both platforms to keep conversations consistent.

🧩

Rich Content Support

Supports messages, replies, attachments, stickers, embeds, join messages, and polls.

🪝

Webhook Bridging

Messages are bridged using webhooks so they appear with the original sender’s name and avatar.

⚙️

Flexible Deployment

Get started instantly using our hosted bot or run the bridge yourself with Docker for full control.
