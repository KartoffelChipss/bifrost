# Getting Started with Bifröst

Welcome to the Bifröst documentation! This guide will help you get started with Bifröst, the bridge between Discord and Fluxer.

You have two options to get started:

1. Use the **hosted instance** of Bifröst, which is the quickest way to get up and running without any technical setup: [Using the Hosted Instance of Bifröst](/guide/hosted-instance)
2. **Self-host** Bifröst using Docker, which gives you full control over your deployment and allows for customization: [Self-Hosting Bifröst](/self-hosting)
