# Using the Hosted Instance of Bifröst

If you don't want to self-host Bifröst, you can use the hosted instance provided by the developer. This is a great option if you want to get started quickly without worrying about server setup and maintenance.

::: warning
The hosted instance currently may be unavailable, due to Discord requiring a verification for bots in over 100 servers and Discord taking a long time to review the bot.

We are working on getting the bot verified as soon as possible, but in the meantime, you can use the self-hosting guide to run Bifröst on your own server.
:::

## 1. Add the bots to your Discord and Fluxer servers

To use Bifröst, you have to add both the Discord and Fluxer bots to your servers:

- [Add to Discord](https://discord.com/oauth2/authorize?client_id=1475436995697180845&permissions=536947712&integration_type=0&scope=bot)
- [Add to Fluxer](https://web.fluxer.app/oauth2/authorize?client_id=1475208219145040215&scope=bot&permissions=536947712)

## 2. Link Your Servers

After adding the bots, you're all set up and can follow the instructions in the [Linking Guide](/guide/linking) to link your Discord and Fluxer servers and channels together.
