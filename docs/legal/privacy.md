# Bifröst Privacy Policy

**Last Updated:** March 7, 2026

## 1. Overview

Bifröst is designed with minimal data collection. The bot only stores the information necessary to operate its server and channel bridging functionality.

## 2. Data Collected

Bifröst may store the following Discord identifiers:

- **Server IDs** — when servers are linked
- **Channel IDs** — when channels are linked between servers
- **Message IDs** — used for message deletion propagation across bridged channels

## 3. Data Not Collected

Bifröst does **not** store:

- Message content
- Usernames
- User IDs
- Attachments
- Any personal information

## 4. Purpose of Data Storage

Stored data is used solely for the following purposes:

- Maintaining links between servers
- Maintaining links between channels
- Synchronizing message deletions across bridged channels

## 5. Data Retention

Data is retained only for as long as necessary for the bot to function.  
If a bridge is removed or the bot is removed from a server, associated data may be deleted.

## 6. Data Sharing

Bifröst does **not sell, share, or distribute** stored data to third parties.

## 7. Security

Reasonable measures are taken to protect stored data. However, no method of electronic storage or transmission over the internet is completely secure.

## 8. Changes to This Policy

This Privacy Policy may be updated from time to time. Continued use of the bot constitutes acceptance of any updates.

## 9. Self-Hosted Instances

Bifröst is open-source software and may be self-hosted by third parties.

This Privacy Policy applies only to the **official instance of Bifröst operated by the developer**.

If you interact with a self-hosted instance, the operator of that instance is responsible for any data storage or processing performed by their deployment.
